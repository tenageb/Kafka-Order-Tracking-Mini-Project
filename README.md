# Kafka-Order-Tracking-Mini-Project
Dockerized Kafka mini project for simulating order production and real-time order tracking using Python producer and consumer scripts.

This project is a simple real-time data-streaming mini-project built with Python, Apache Kafka, and Docker. It includes a producer script that generates order messages in JSON format and sends them to a Kafka topic, and a consumer or tracker script that reads those messages in real time and displays the order details. The project is useful for practicing core Kafka concepts, including topics, producers, consumers, consumer groups, message serialization, and real-time event processing, in a containerized environment.
